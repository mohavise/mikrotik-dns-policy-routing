# Microsoft Azure Sources

Official sources:

- https://azure.microsoft.com/
- https://learn.microsoft.com/en-us/azure/azure-portal/azure-portal-safelist-urls?tabs=public-cloud

Scope covers the Azure website, public-cloud portal framework, Resource Manager, Microsoft Entra authentication, and Microsoft Graph dependencies documented for portal operation.

Azure workload, storage, database, application-hosting, CDN, and public service IP ranges are intentionally excluded.
