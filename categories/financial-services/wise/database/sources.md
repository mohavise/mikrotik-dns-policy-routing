# Wise Sources

Official sources:

- https://wise.com/
- https://docs.wise.com/guides/developer/environments

The official Wise site uses `wise.com` for its consumer banking and money-transfer service. Official Wise Platform environment documentation identifies `api.wise.com` and `api-mtls.transferwise.com` as production endpoints, so the list includes both `wise.com` and the legacy `transferwise.com` domain families. Generated rules cover each root domain and all of its subdomains.

Wise sandbox domains and shared banking, cloud, CDN, analytics, and app-store infrastructure are intentionally excluded.
